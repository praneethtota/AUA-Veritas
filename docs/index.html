Install on macOS

1

Download the .dmg

Click the download button above. The installer is not signed with an Apple Developer certificate — this is normal for open-source apps distributed outside the App Store.

2

Open the .dmg and drag to Applications

Double-click AUA-Veritas-*.dmg, then drag the app icon to your Applications folder.

3

Open the app (first launch only)

Right-click AUA-Veritas in Applications → click Open → click Open in the dialog. You only need to do this once.

Or: System Settings → Privacy & Security → scroll down → click Open Anyway next to AUA-Veritas.

4

Connect your AI models

Enter your API keys. Keys are stored in macOS Keychain — never uploaded anywhere.

Need free options? Google Gemini and Groq (Llama) both have generous free tiers.

Why the security warning? Apple requires a $99/year Developer account to sign apps for distribution. AUA-Veritas is free and open-source, so we distribute directly from GitHub. The right-click → Open method bypasses Gatekeeper for apps you trust. The source code is fully open at github.com/praneethtota/AUA-Veritas.
